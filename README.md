# 🐱 MichiApp

## 📌 Descripción del proyecto

MichiApp es una aplicación móvil desarrollada en Android utilizando Jetpack Compose, que simula una red social enfocada en gatos. La aplicación permite navegar entre diferentes pantallas donde el usuario puede visualizar un feed de publicaciones gatunas, interactuar con ellas mediante un sistema de calificación por estrellas y consultar un perfil personalizado.

El flujo de la aplicación está compuesto por múltiples pantallas (Inicio, Feed, Detalle y Perfil), conectadas mediante un sistema de navegación funcional. Se implementan componentes interactivos como botones, listas dinámicas y elementos clicables, ofreciendo una experiencia simple pero intuitiva para el usuario.

## 🛠 Tecnologías utilizadas

* **Kotlin**

* **Material Design 3 (Material You)**

* **Android Studio**

---

## ▶️ Instrucciones para ejecutar la app

1. Abrir el proyecto en Android Studio.
2. Asegurarse de que Gradle sincronice correctamente las dependencias.
3. Ejecutar la aplicación en:

   * Un emulador de Android, o
   * Un dispositivo físico conectado.
4. Navegar entre las pantallas utilizando los botones disponibles:

   * Inicio → Feed
   * Feed → Detalle
   * Detalle → Perfil
   * Perfil → Inicio
